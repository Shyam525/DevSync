# DevSync

This repository has been updated to include a structured monorepo layout with:

- `packages/client` for the React + Vite front-end
- `packages/server` for the Express + Socket.IO back-end
- `packages/shared` for shared types and constants
- `docs/` for API and architecture documentation
- `scripts/` for local development automation
- `.github/workflows/` for CI and deploy workflows
